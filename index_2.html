Key Terms From Chapter 2

  1. External Sheet Style

    2. An external style sheet saved as a separate text document but linked to from the webpage it modifies.

  2. ID selector

    3. A specific type of Cascading Style Sheets selector intended to be used once on a webpage. It begins with the # symbol.

  3. Inline Style

    4. A formatting declaration created in the HTML tag being formatted.

  4. List Item

    5. One item inside an ordered or unordered HTML list.

  5. Nested

    6. HTML tags that are created between other HTML tags.

  6. Ordered List

    7. HTML tag set used to create a numbered or lettered list of similar items, in order.

  7. Relative links

    8. Links that do not contain the full file path to a webpage.

  8. Unordered List

    9. HTML tag set used to create a bulleted list of items.

  9. Version Control

    10. Software that is used to create date-time stamped copies of coding projects.

  10. WYSIWYG

    11. An editor that shows the user "What You See Is What You Get."
